# Portfólio refinado — Vinicius Costa (PT/EN + Light/Dark)

## Como rodar
- Abra `index.html` no navegador
- (Opcional) use Live Server no VS Code

## Colocar sua foto
1) Adicione sua foto em: `assets/profile.jpg`
2) No `index.html`, troque o `src` do placeholder para `assets/profile.jpg`

## Colocar prints reais (recomendado)
- `assets/projects/controle-financeiro.jpg`
- `assets/projects/site-luca.jpg`

Depois, no `index.html`, troque os `src` dos placeholders para esses JPGs.

## Links do GitHub
No `index.html`, substitua os `href="#"` dos botões "Ver código (GitHub)" pelo link real do repo.
